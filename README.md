#  Spotify Genre Classification

Prédiction du **genre musical** à partir des **features audio Spotify**, dans le cadre d’un projet de Data Science (ENSAE).

---

##  Installation

`pip install -r requirements.txt`

---

##  Utilisation

1. Ouvrir les notebooks dans le dossier **notebooks** :
   - 01_exploration.ipynb  
   - 02_preprocessing.ipynb  
   - 03_modelisation.ipynb

2. Le code du projet se trouve dans le dossier **src** :
   - preprocessing.py  
   - models.py  
   - evaluation.py  

3. Les résultats et figures sont enregistrés dans le dossier **reports**.

---

##  Structure du projet

spotify-genre-project/
├─ Data/
